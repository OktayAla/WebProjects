Projelerim

AI Destekli Borsa Analiz Uygulaması

Python ile geliştirilen bu proje, Tensorflow ve Sklearn kütüphanelerini kullanarak bir makine öğrenmesi modellemesi oluşturur. Hazırladığı model sayesinde borsada analiz ve grafik yorumlaması yapar. Stop noktası, TP noktası, Short ve long noktalarını belirler.

Demo GitHub

Talep Bildirim Sistemi

Kullanıcıların taleplerini ve arızalarını web üzerinden bildirebilecekleri MySQL ve PHP tabanlı bir ticket sistemi. Bir ilçe belediyesi üzerinde pilot olarak seçilerek demosu hazırlanmıştır. İlgili görevli ticket oluşturduktan sonra takibini ve durumunu kontrol edebilmekte. İdareci ise bu ticketler üzerinde iş takibini grafiksel olarak raporlarını çekebilmektedir.

Demo GitHub

Nöbetçi Eczaneler

Denizli ili için tasarlanan bu program, C# ile hazırlanmıştır. Web sitesi üzerinden anlık olarak Merkezefendi ve Pamukkale ilçelerinde o güne ait nöbetçi eczanelerin verilerini çekmekte. GUI olarak görselleştirilmiş olup hastane kullanımı için hazırlanmıştır.

Demo GitHub

YouTube MP4 Downloader

Kullanıcıların YouTube linklerini yapıştırarak MP3 formatında indirme yapabilmesini sağlayan basit bir araçtır. Python ve CustomTkinter ile geliştirilen uygulama, yt-dlp kütüphanesi sayesinde videoları masaüstüne MP3 olarak kaydeder. Kullanıcı dostu arayüzüyle tek tıkla indirme yapabilmektedir.

Demo GitHub

Uzay Savaş Oyunu

JavaScript ve HTML5 Canvas kullanılarak geliştirilmiş bir uzay temalı shoot 'em up oyunudur. Oyuncu, ekranın alt kısmında hareket eden bir uzay gemisini kontrol ederek yukarıdan gelen düşman gemilerine, asteroidlere ve uydulara ateş eder.

Demo GitHub

İnsan Kaynakları Yönetim Sistemi

Bu proje, şirketlerin personel yönetim süreçlerini dijitalleştirmek için geliştirilmiş bir İnsan Kaynakları Yönetim Sistemi (HRMS) uygulamasıdır. HTML, CSS, JavaScript ve PHP kullanılarak oluşturulmuştur. Temel olarak veritabanı yerine tüm veriyi json olarak yedekler.
